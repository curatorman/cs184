Assignment 2: Ray Tracer
For this assignment, we created a ray tracer.
Unless specified, the time to render each image was < 2 seconds.
All command line invocations were simply ./as2, and we manually toggled on and off tests that we made in test.cpp.
Animations were created by generating multiple images and then manually putting them together to create videos.
